ID Card Management System

Manage student and employee ID cards with QR codes, barcodes, and PDF export

📋 Profiles API

CRUD operations for user/student/employee profiles

GET /profiles - List all profiles
GET /profiles/{id} - Get profile by ID
GET /profiles/uuid/{uuid} - Get profile by UUID
GET /profiles/search?name=&department=&type= - Search profiles
POST /profiles - Create profile
PUT /profiles/{id} - Update profile
DELETE /profiles/{id} - Delete profile
POST /profiles/{id}/photo - Upload photo
GET /profiles/{id}/photo - Download photo

🆔 ID Cards

Live preview, PDF export, QR codes, and barcodes

GET /id-cards/preview/{id} - Live preview ID card
GET /id-cards/pdf/{id} - Download PDF
POST /id-cards/pdf/batch - Batch PDF generation
GET /id-cards/qr/{profileId} - Get QR code image
GET /id-cards/barcode/{profileId} - Get barcode image

🎨 Templates API

Manage ID card templates (colors, organization name, tagline)

GET /templates - List all templates
GET /templates/{id} - Get template by ID
GET /templates/search?name= - Search templates
POST /templates - Create template
PUT /templates/{id} - Update template
DELETE /templates/{id} - Delete template

🚀 Quick Start

  1. Create a profile: curl -X POST http://localhost:8091/profiles -H "Content-Type: application/json" -d '{"fullName":"John Doe","title":"Engineer","type":"STUDENT"}'
  2. View ID card: http://localhost:8091/id-cards/preview/1
  3. Download PDF: http://localhost:8091/id-cards/pdf/1
